# The Content Steward's Playbook

## Steward Responsibilities

- Maintain quality
- Govern metadata
- Monitor performance
- Improve discoverability
- Archive outdated assets
- Drive continuous improvement

## Key Questions

- Is this content still accurate?
- Is it discoverable?
- Does it duplicate existing content?
- Who owns it?
- When should it be updated?
- When should it be archived?
